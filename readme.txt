Plugin to display the live scores of WvW matchups in Guild Wars 2. Working with PhpGw2Api and data from the API developed by ArenaNet.

== Description ==

Guild Wars 2 - WvW Matchups, is a simple plugin to display the live scores of WvW matchups in Guild Wars 2.

Very simple to use, simply create a WIDGETS and drag it to your sidebars and select your favourite world!

The plugin refreshes its data by ArenaNet API (beta) using the SDK PhpGw2Api (PhpGw2Api is a PHP wrapper for the GW2 API.)

https://forum-en.guildwars2.com/forum/community/api/API-Documentation

== Installation ==

1. Upload the entire `gw2-wvw-matchups` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. You will find 'GW2 - WvW Matchups' in your Widgets page on WordPress admin panel.
1. Drag the widget to where you want to have and set the world who want to appear to show the match with scores.
